# 📖 About

This repository contains a collection of interactive HTML pages designed to explain key industrial functional safety concepts in a simple and visual way.

Instead of reading long theoretical notes, each topic is presented as a standalone interactive webpage that can be opened directly in a web browser or viewed online using GitHub Pages.

The objective is to make industrial safety concepts easier to understand while building a reusable learning resource for students, engineers, and anyone preparing for automation interviews.

---

# 🎯 Learning Objectives

This repository aims to explain:

- The difference between a Normal Stop and an Emergency Stop
- Safety Integrity Level (SIL)
- Safety Instrumented Function (SIF)
- Standard PLC vs Safety PLC
- The role of functional safety in industrial automation
